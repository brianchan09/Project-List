# Aim
Food stoarge and delivery system.<br>
This program use to save all the delivery record, with search and modify function.
<br>Record will be saved in the `inventory.txt` <br>

# Item data
- Item Name
- Item Number
- Category
- Quantity
- Weight
- Recipient
- Final Destination
- Delivery Status


# How to Run
Open start.bat<br>
type "ICP v6" and press ENTER to run
